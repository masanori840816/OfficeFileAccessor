# OfficeFileAccessor
## About
This is an ASP.NET Core application to access docx and xlsx files.

### Environtments
* .NET ver.9.0.100
* Node.js ver.22.11.0
* React

### Build
#### For Windows
```
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true
```
